# H2 Database 란?

  - 오픈 소스 데이터베이스이며 Java로 작성되었습니다.  
  - 매우 빠르고 매우 작은 크기입니다.  
  - 주로 메모리 내 데이터베이스로 사용됩니다.  
  데이터를 메모리에 저장하고 디스크에 데이터를 유지하지 않습니다.  
  - 데이터를 유지해야하는 경우 디스크에 데이터를 유지할 수도 있습니다.
